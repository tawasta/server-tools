===============================
Read Access Group for Invoicing
===============================

* Adds a new group that is allowed to read invoicing model data
* Intended for situations where employees should have additional read-only 
  access rights to invoicing

Configuration
=============
* Add the user of your choice to the new "Read access to invoicing" group
