=============
Auth Odoo IdP
=============

This modules intention is to enable returning data regarding logged in user to SSO systems (Odoo as IdP master). SSO systems authenticate by using XML-RPC with common.authenticate -method.
Then they can fetch that user's data using XML-RPC procotol by calling res.users user_info_json -method.
