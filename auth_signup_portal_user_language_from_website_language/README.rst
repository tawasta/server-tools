=======================================================
Auth Signup: Portal User Language from Website Language
=======================================================

* When portal user registers via website, their language
  is set based on what the website language selector
  has stored in context.
* This overrides the default behaviour where the user
  language would be pulled from the "Portal User"
  res.users record.
