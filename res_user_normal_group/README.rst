=================
User Normal Group
=================

Adds a new group, Normal User, to available groups for Odoo. It is
meant for hiding menu items that Internal User has an access to.

Only use this module if you are sure about what you are doing! It will
hide some menu items from users, after launching its inactive scheduled
action, unless they belong to Normal User group.
